## Akses Category
#### Mengambil data seluruh category
```
method : GET
Url : /category
```
#### Menambahkan data category
```
method : POST
Url : /category
```
#### Mengubah data category
```
method : PUT atau PATCH
Url : /category/{category_id}
```
#### Menghapus data category
```
method : DELETE
Url : /category/{category_id}
```
